⚠️ COMMON ISSUES:

"Python not found"
  → Install Python from python.org
  → Add to PATH during installation

"pip not found"
  → Use: python -m pip install -r requirements.txt

"Port 5000 in use"
  → Edit app.py line 185: app.run(debug=True, port=5001)

"Database error"
  → Delete data.db file
  → Restart server

"Styles not loading"
  → Press Ctrl+F5 to hard refresh
