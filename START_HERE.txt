⚠️ TROUBLESHOOTING

"Python not found"
  → Install Python from python.org
  → Make sure Python is in PATH

"pip not found"
  → Use: python -m pip install -r requirements.txt

"Port 5000 already in use"
  → Edit app.py and change port number: app.run(debug=True, port=5001)

"Database error"
  → Delete data.db file and restart server

"Styles not loading"
  → Press Ctrl+F5 to hard refresh browser
